# Maibot-personality_switch_plugin
# 插件功能总结
1. 核心突破：解决单一maibot局限，实现“单bot多人格”效果，无需额外开启多个maibot
2. 人格交互：支持多个人格聊天、互动；默认保持当前人格，输入其他人格名/命令可切换
3. 人格增强体系：含成就系统、人格专属技能，以及人格动态关系与成长（关系升级、能力解锁）
4. 实用功能集成：提供待办、天气查询、翻译、SD绘图、语音等第三方工具
5. 灵活适配能力：支持人格热插拔、多场景适配，可智能化交互（含意图+情绪识别）
6. 管理与配置：细粒度权限管理（管理员/普通用户/游客）、Web端可视化配置工具、一键备份/迁移（本地+云）
7. 辅助与记忆：配备监控面板（统计活跃度、数据备份），支持短期/长期/重要记忆存储
8. 该插件会持续更新，一般是优化人格的功能，或者是增添新的工具在里面，也欢迎大家投稿新功能和优化的地方。
